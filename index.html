Tic-Tac-Toe

Web-Based Game

HTML // CSS // JavaScript // JQuery // AJAX

For this project, I used the classic and iconic game to learn how to take raw JavaScript code and connect it to the web browser for real humans to use. As a lifetime player of video games, it was really fun to actually build my own and learn more about what goes on behind the scenes.

This front-end application was built using JavaScript code and utilizing the power of the JQuery library to render HTML in the DOM. I also used AJAX to consume a pre-built, REST API.

What’s Next: I would like to implement other features of the provided API, which includes the ability to save game states and challenge other players over the internet. I was also hoping to provide a way for the user to customize the game by setting the size of the grid and the number of rounds they would like to play.

OkFido

Pet Adoption Database

HTML // CSS // JavaScript // JQuery // AJAX // Handlebars // Ruby on Rails // PSQL

Using my passion for dogs as the subject, I explored building a full-stack application from the ground up. The name OkFido is inspired by the popular online dating site OkCupid, with the concept of being a website that will match owners with the perfect dog, and vice versa. Currently, you can create an account and search a database of adoptable dogs, and save your favorites to refer to later.

The application is built using JavaScript and AJAX on the front end that consumes a custom Rails API.

What’s Next: Soon I plan to improve the search functionality of the site by improving the filtering mechanisms. I also envision a real life application of this concept by making it easier for shelters to screen potential candidates by matching users and dogs based on a variety of important criteria.

Awesome CMS

Content Management System

HTML // CSS // JavaScript // JQuery // AJAX // Handlebars // Express // Mongo DB // Mongoose

Awesome CMS is a project where I was able to experience developing with others as a team. We were tasked with creating a WordPress-like content management system that allows an administrator to create, edit, and delete blog posts and pages. This was an incredible experience, as we were able to put our minds together and solve a hard problem.

This full-stack application was built using Express with Mongo DB and Mongoose to create a RESTful API. We used AJAX to consume the back-end, and utilized Handlebars to render this data on the front-end.

What’s Next: Through this process, we learned about the security consequences of allowing a user to directly write HTML onto the page, enabling them to inject any type of malicious JavaScript they choose. Fixing this will be the next step in the progress of the project along with cleaning up the user interface.

Gather

Casual Social Network

HTML // CSS // JavaScript // JQuery // AJAX // Handlebars // Ruby on Rails // PSQL

Gather is a social media application that allows people to invite friends to casual events created on the fly. The concept is to be a less formal alternative to the Facebook event creation feature, where users can create, and invite friends to casual get-togethers, (gatherings) on a whim.

This project helped me become more comfortable building full-stack web applications using Rails, while building a responsive, user friendly front-end with JavaScript, AJAX, and Handlebars.

What’s Next: I plan to add more features as this project ages, such as the ability to add checklists to a gathering, containing items/tasks that are needed to make the event happen. These items can be checked off by each person invited, allowing the host to know who is doing/bringing what.

I would also like to integrate Ember into this project to streamline the front-end, and to create an even more responsive experience.
